Notes

  • This is a TinyMCE dependent plugin
    • Only tested on TinyMCE 5+
  • Unfortunately, this is not currently compatible with mobile devices, so the user is redirected to another page to handle it.
  • The style of the edited text depends on your stylesheet files.
